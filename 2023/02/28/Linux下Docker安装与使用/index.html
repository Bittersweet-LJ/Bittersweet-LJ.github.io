修改镜像标签

由于镜像名称较长，您可以修改镜像标签以便记忆区分。

新建并运行容器

e1abc是执行docker images命令查询到的ImageId，使用docker run命令新建容器。

退出容器。使用exit命令退出当前容器。

run命令加上–d参数可以在后台运行容器，--name指定容器命名为apache。

进入后台运行的容器。

将容器做成镜像，命令的参数说明：docker commit <容器ID或容器名> [<仓库名>[:<标签>]]。

例如：docker commit 4c8066cd8**** apachephp:v1

运行容器并将宿主机的8080端口映射到容器里去，就可以通过ECS实例IP地址加8080端口访问测试
